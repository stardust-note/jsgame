# 플러피 새 (jsgame)

간단한 플러피 버드 스타일의 HTML5 캔버스 게임입니다. 스페이스바나 클릭으로 새를 날게 하여 파이프 사이를 통과하며 점수를 얻을 수 있습니다.

## 실행 방법

1. 저장소를 클론하거나 다운로드합니다.
2. `index.html` 파일을 웹 브라우저로 열면 바로 게임을 플레이할 수 있습니다.

## 조작법

- **스페이스바 / 위쪽 화살표**: 새가 위로 날아오릅니다.
- **마우스 클릭 또는 터치**: 동일하게 새가 위로 날아오릅니다.

## 기능

- 로컬 최고 점수 저장 (브라우저 `localStorage` 사용)
- 부드러운 애니메이션과 배경 그래픽
- 한국어 UI 및 안내 문구
